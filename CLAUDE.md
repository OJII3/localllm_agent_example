# コーディングエージェントセミナー資料

## ターゲット

- 情報学科の研究室の学生
- 開発経験は無~多少まで
- CLI は普段使いしない人
- Git は普段使いしない人
- Windows ユーザーと Mac ユーザーがいる。Windows ユーザーは WSL が入っている
- 普段使用しているAIは人による。使用していない人もいる

## 最低要件

- pptx で登壇資料が完成している
- 無料でできる範囲で完結している
- 最大1時間で終わる程度の内容になっている
- 作者の都合により、このリポジトリ自体の環境構築には Nix, direnv を用いる(講習内では使用しない)

## 目的

- コーディングエージェントとはどんなものかを理解する
- ある程度手を動かして、Web上のチャットに留まらない範囲の自動化ができることを体験する


## 執筆方針

1. 次のセクションにあるセミナー内容について、上記ターゲット・最低要件・目的に沿うよう修正提案・代替案の提案をする
2. pptx 操作のための skills を導入する. セミナー内容で扱う題材に適した(特に、高性能とは言えないモデルが操作する上で有用な) skills を選定する
3. pptx 資料を作成する。
